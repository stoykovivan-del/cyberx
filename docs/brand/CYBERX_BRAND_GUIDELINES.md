# CyberX Brand Guidelines

## Overview
CyberX builds autonomous software systems that create, operate, and sustain profitable products with minimal human involvement. The brand should convey innovation, intelligence, automation, and reliability.

---

## Color Palette

### Primary Colors
| Name | Hex | Usage |
|------|-----|-------|
| Deep Navy | `#0A0E1A` | Primary backgrounds, footer |
| Electric Blue | `#0066FF` | Primary buttons, links, accents |
| Pure White | `#FFFFFF` | Text on dark, card backgrounds |

### Secondary Colors
| Name | Hex | Usage |
|------|-----|-------|
| Slate Gray | `#64748B` | Secondary text, borders |
| Light Gray | `#F1F5F9` | Light backgrounds, dividers |

### Accent Colors
| Name | Hex | Usage |
|------|-----|-------|
| Cyan Glow | `#00D4FF` | Highlights, gradients, CTAs |
| Success Green | `#10B981` | Success states, positive indicators |
| Warning Amber | `#F59E0B` | Warnings, attention |

---

## Typography

### Font Families
- **Primary Font:** Inter (Google Fonts) - clean, modern, excellent readability
- **Monospace Font:** JetBrains Mono - for code/technical elements

### Font Sizes
| Element | Size | Weight |
|---------|------|--------|
| H1 (Hero) | 56px / 3.5rem | 700 |
| H2 (Section) | 40px / 2.5rem | 600 |
| H3 (Subsection) | 28px / 1.75rem | 600 |
| Body | 16px / 1rem | 400 |
| Small | 14px / 0.875rem | 400 |
| Caption | 12px / 0.75rem | 400 |

### Line Heights
- Headings: 1.2
- Body: 1.6

---

## Logo Guidelines

### Logo Variants
- **Full Logo:** "CyberX" wordmark with icon (dark background)
- **Icon Only:** For favicon and small spaces (32x32 minimum)
- **Light Variant:** White text for dark backgrounds

### Clear Space
- Maintain minimum 24px clear space around logo
- Never place logo on busy backgrounds without container

### Minimum Sizes
- Digital: 32px height
- Print: 20mm height

## Hero Section Messaging

### Value Proposition
**Primary Headline:**
"Autonomous Systems That Build, Run, and Scale Your Business"

**Subheadline:**
"CyberX creates intelligent software that operates independently—building products, managing operations, and driving growth without manual intervention."

### Key Messages
1. **Automation First** - "Minimize human involvement, maximize results"
2. **Intelligent Operations** - "Systems that think, learn, and adapt"
3. **Profitable by Design** - "Built for sustainable business growth"

### Call to Action
- Primary: "Get Started" or "Build with CyberX"
- Secondary: "View Documentation" or "Talk to Sales"

---

## Design Principles

1. **Clean & Modern** - Minimal clutter, purposeful whitespace
2. **Tech-Forward** - Reflects autonomous/AI capabilities
3. **Trustworthy** - Professional, reliable appearance
4. **Performance-Oriented** - Fast, efficient, optimized

---

## Usage Notes
- Use cyan glow accents sparingly for visual interest
- Apply gradients subtly (avoid overwhelming)
- Ensure high contrast for accessibility (WCAG AA minimum)
- Dark theme preferred for hero sections
